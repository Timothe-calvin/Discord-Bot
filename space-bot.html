Add to Discord

About This Bot

Space Bot is a versatile multipurpose Discord bot that offers a wide range of features including moderation tools, utility commands, and fun activities. With an easy-to-use command structure, it helps server administrators manage their communities effectively while providing entertainment features for members.

Bot Features

  • Moderation commands
  • Message management
  • Weather information
  • AI chat capabilities
  • Fun games and activities
  • Quotes and inspirational content

Getting Started

Space Bot uses the + prefix for all commands. To get started, type +help in any text channel to see a list of available commands.

The bot requires certain permissions to function properly, including message management, member management, and embed links. Make sure to grant these permissions when adding the bot to your server.

Bot Commands

General Commands
+help

Shows the help message with all available commands

+help
+quote

Get an inspirational quote

+quote
+joke

Get a random joke

+joke
+fact

Get a random fun fact

+fact
Moderation Commands
+kick @user

Kick a user from the server (requires kick permissions)

+kick @username
+ban @user

Ban a user from the server (requires ban permissions)

+ban @username
+clear [number]

Clear a specified number of messages (requires manage messages permission)

+clear 10
Utility Commands
+weather [city]

Get weather information for a specified city

+weather New York
+chat [message]

Chat with the bot's AI assistant

+chat What's the meaning of life?
